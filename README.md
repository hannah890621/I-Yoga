I-Yoga
===

利用 OpenPose 探測指導者的四肢與軀幹的架構，再使用 dtw 進行使用者動作比對，過程中有任何姿勢上不符合的地方系統會發出警訊提醒，並協助調整到正確的姿勢，降低在家運動復健可能受傷的風險。
